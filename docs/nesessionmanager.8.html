NAME

nesessionmanager —
Daemon responsible for running and managing Network Extension sessions

DESCRIPTION

nesessionmanager is part of the Network Extension framework. It is responsible for starting and stopping Network Extension sessions and for setting up and maintaining network configuration and policy associated with Network Extension sessions.

Types of Network Extension sessions include:

  • VPN
  • Content Filter
