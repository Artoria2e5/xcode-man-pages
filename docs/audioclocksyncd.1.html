NAME

audioclocksyncd — clock management and interface for Precision Time Protocol.

SYNOPSIS

audioclocksyncd

DESCRIPTION

The audioclocksyncd executable is used for the management of Precision Time Protocol (IEEE 1588) clocks and the interface between applications and the clocks.

audioclocksyncd is launched automatically by launched as needed on demand and should not be launched manually.
