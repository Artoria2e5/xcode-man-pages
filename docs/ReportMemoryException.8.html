NAME

ReportMemoryException — generates memory usage diagnostic logs

SYNOPSIS

ReportMemoryException

DESCRIPTION

The ReportMemoryException command is a system service which should only be launched by launchd. It generates memory usage diagnostic logs as indicated by system events such as memory limit violations.
