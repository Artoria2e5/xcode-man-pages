NAME

Photo Library Migration Utility —
Upgrade historical iPhoto/Aperture libraries

DESCRIPTION

Photo Library Migration Utility prepares a clone of an iPhoto/Aperture library for migration to Photos. If required, an upgrade and rebuild will be performed on the cloned library. The utility is for Photos.app use only.
