NAME

microstackshot — aggregates microstackshot call graphs

DESCRIPTION

The microstackshot command aggregates and saves daily microstackshot call graphs to the diagnostic reports directory. The reports may be sent to Apple if the device is opted into data collection.

This command works only within the context of a launchd job and should not be run from the command line.
