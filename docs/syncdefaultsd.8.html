NAME

syncdefaultsd —
NSUbiquitousKeyValueStore

DESCRIPTION

syncdefaultsd is the system daemon backing the NSUbiquitousKeyValueStore feature.

There are no configuration options to syncdefaultsd, and users should not run syncdefaultsd manually.
