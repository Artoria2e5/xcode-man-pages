NAME

usbmuxd — iOS communications daemon

SYNOPSIS

usbmuxd [options]

DESCRIPTION

The usbmuxd daemon manages communications between MacOS hosts and iOS devices. USB and WiFi (via Bounjour) connections are supported.

usbmuxd is started automatically at boot time and does not require user or operator intervention.

OPTIONS

--version
Print version number and exit
--launchd
Passed by launchd to inform usbmuxd that it is being launched as a daemon
--debug n
Set debug level
