NAME

umtool — Diagnostic tool for UserManagement

SYNOPSIS

umtool sysdiagnose

DESCRIPTION

Use umtool to output diagnostic data for troubleshooting UserManagement.

Available commands:

sysdiagnose
Outputs diagnostic data for collection by sysdiagnose(1)
