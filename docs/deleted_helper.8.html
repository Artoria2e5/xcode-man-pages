NAME

deleted_helper — The APFS purge daemon.

DESCRIPTION

deleted_helper is a system daemon that purges APFS Purgeable files on demand. deleted_helper listens for purge messages from CacheDelete's daemon (deleted). deleted_helper is not intended to be invoked directly.
