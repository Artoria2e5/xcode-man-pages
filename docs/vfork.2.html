NAME

vfork — deprecated system call to create a new process

SYNOPSIS

#include <unistd.h>

pid_t
vfork(void);

DESCRIPTION

The vfork system call can be used to create new processes. As of macOS 12.0, this system call behaves identically to the fork(2) system call, except without calling any handlers registered with pthread_atfork(2).

This system call is deprecated. In a future release, it may begin to return errors in all cases, or may be removed entirely. It is extremely strongly recommended to replace all uses with fork(2) or, ideally, posix_spawn(3).
