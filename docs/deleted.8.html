NAME

deleted —
The purgeable space / cache management daemon.

DESCRIPTION

deleted is a system daemon that keeps track of purgeable space via registered services. deleted listens for low-space events from the file system and attempts to avoid running out of disk space by requesting that clients purge space. deleted is not intended to be invoked directly.

FILES

/System/Library/CacheDelete
CacheDelete service definitions provided by Apple.
