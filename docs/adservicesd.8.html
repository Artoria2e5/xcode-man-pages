NAME

adservicesd —
advertising services daemon

DESCRIPTION

adservicesd is a daemon that serves advertisements to selected Apple first-party applications, and provides anonymous analytics data to Apple which may be shared with advertisers. No personally identifiable information is collected or shared with any third parties.

There are no options for adservicesd, and users should not run adservicesd manually.
