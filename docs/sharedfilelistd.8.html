NAME

sharedfilelistd —
User agent which manages lists of file system items

DESCRIPTION

sharedfilelistd is a per-user agent that manages lists of recent and favorite documents, applications, and volumes. It stores the lists for global Apple Menu > Recent Items, the per-application File > Open Recent, and some of the content in the Finder sidebar.
